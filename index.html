Projeto em Andamento

Minha Rede Social

Descrição do Projeto: A "Minha Rede Social" é uma plataforma simples de mídia social desenvolvida para permitir que os usuários compartilhem mensagens e interajam entre si. O projeto é implementado utilizando tecnologias web como HTML, CSS e JavaScript. A seguir está um resumo das principais funcionalidades do site: Login e Registro de Usuário: Os usuários podem fazer login com suas credenciais existentes ou registrar uma nova conta. Postagem de Mensagens: Os usuários autenticados podem criar novas postagens digitando texto em um campo de entrada e clicando no botão "Postar". As postagens são exibidas em uma lista na página principal. Logout: Os usuários podem sair de suas contas clicando no botão "Logout". Isso os desconecta da plataforma e os redireciona para a tela de login.
